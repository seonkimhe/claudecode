# 날씨 앱 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | 날씨 앱 |
| 한 줄 소개 | 도시 이름 검색 또는 현재 위치로 실시간 날씨와 5일 예보를 확인하는 웹 앱 |
| 목적 | 별도 설치나 로그인 없이, 브라우저에서 즉시 현재 날씨와 앞으로의 예보를 확인 |

## 2. 배경 및 목적

기존 날씨 앱/서비스는 회원가입, 위치 권한 강제, 광고 등으로 인해 "지금 여기 날씨"를 빠르게 확인하려는 사용자에게 불필요한 단계를 요구하는 경우가 많다. 본 프로젝트는 빌드 과정이나 서버 인프라 없이 정적 파일만으로 동작하는 가벼운 날씨 조회 도구를 목표로 한다. API 키 발급이 필요 없는 무료 공개 API(Open-Meteo)를 사용해 진입 장벽을 최소화한다.

## 3. 타겟 사용자

- 한국어를 사용하는 일반 사용자
- 특정 도시(예: 서울, 부산 등)의 날씨를 빠르게 검색하고 싶은 사용자
- 현재 위치 기준으로 날씨를 확인하고 싶은 사용자

## 4. 주요 기능

| 기능 | 설명 |
|---|---|
| 도시 이름 검색 | 검색창에 도시명(한글 가능)을 입력하면 좌표로 변환 후 해당 지역 날씨 조회. 서울/부산/대구 등 주요 광역시는 별도 별칭 매핑을 통해 동명의 소도시와 혼동되지 않도록 보정 |
| 현재 위치 조회 | 검색창 옆 📍 버튼 클릭 시 브라우저 위치 정보(`navigator.geolocation`)를 이용해 현재 위치의 날씨 조회 |
| 현재 날씨 표시 | 지역명, 조회 시각, 날씨 아이콘, 현재 기온, 날씨 설명(맑음/흐림 등), 체감온도, 습도, 풍속 표시 |
| 5일 예보 | 오늘 포함 5일간의 날씨 아이콘과 최고/최저 기온을 카드 형태로 표시 |
| 상태 안내 | 검색 중 / 위치 확인 중 문구 표시, 지역을 찾지 못하거나 위치 정보를 가져올 수 없을 때 에러 메시지 표시 |

## 5. 화면 구성

한 페이지(단일 뷰)로 구성되며 상단부터 아래 순서로 배치된다.

1. **헤더** — 타이틀("날씨")과 검색 폼(도시 입력창, 검색 버튼, 현재 위치 버튼 📍)
2. **현재 날씨 카드** — 지역명 · 조회 시각 · 날씨 아이콘/기온(큰 글씨) · 날씨 설명 · 체감온도/습도/풍속 3분할 정보
3. **5일 예보 카드** — 요일별로 아이콘 · 최고/최저 기온을 나란히 배치한 리스트

디자인 컨셉은 하늘을 연상시키는 파란색 계열 그라디언트 배경 위에, 반투명 유리 재질 느낌의 글래스모피즘(glassmorphism) 카드를 올리는 형태다. 모바일 화면을 기준으로 한 단일 컬럼 레이아웃(최대 너비 480px)이다.

## 6. 기술 스택

| 구분 | 내용 |
|---|---|
| 프론트엔드 | 순수 HTML / CSS / JavaScript (프레임워크·번들러·빌드 과정 없음) |
| 서버 | Node.js 내장 `http` 모듈로 구현한 정적 파일 서버 (`server.js`, 기본 포트 8080) |
| 외부 API | [Open-Meteo](https://open-meteo.com) 지오코딩 API(도시명 → 좌표) 및 예보 API(현재 날씨 + 일별 예보). 별도 API 키 불필요, 무료 사용 |
| 데이터 저장 | 없음 (서버·DB 없이 매 요청마다 외부 API 실시간 호출) |

## 7. 데이터 흐름

1. 사용자가 도시명을 입력하고 검색(또는 📍 버튼으로 현재 위치 사용)
2. **지오코딩** — 도시명을 좌표(위도/경도)로 변환. 여러 결과가 있으면 인구가 가장 많은 지역을 우선 선택
3. **날씨 조회** — 변환된 좌표로 현재 날씨 및 5일치 일별 예보 데이터를 요청
4. **화면 렌더링** — 응답받은 데이터를 지역명, 기온, 날씨 아이콘 등 화면 요소에 반영해 표시
5. 페이지 최초 진입 시에는 기본값으로 "서울" 날씨를 자동 표시

## 8. 향후 개선 아이디어 (제안)

> 아래 항목은 현재 구현되어 있지 않으며, 추후 확장을 고려할 수 있는 제안 사항이다.

- 시간대별(1시간 단위) 상세 예보 추가
- 자주 찾는 도시를 즐겨찾기로 저장 (예: `localStorage` 활용)
- 다국어(영어 등) UI 지원
- 다크모드 테마 지원
- 미세먼지 등 대기질 정보 추가
- 오프라인 상태 대응(마지막 조회 결과 캐싱)
