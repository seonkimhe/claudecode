# 쇼핑몰 웹앱 계획

학습/연습용 쇼핑몰. 실제 백엔드 없이 목업(mock) 데이터로 동작. `learn-playwright/weather-app`과 동일한 순수 HTML/CSS/JS + `server.js` 정적 서버 패턴을 따른다.

## 파일 구성

```
shopping-mall/
├── index.html      # 상품 목록 + 장바구니 패널 + 체크아웃 폼 (한 페이지)
├── style.css
├── script.js       # 상품 데이터, 렌더링, 장바구니 로직
└── server.js       # 정적 서버 (weather-app/server.js 재사용, 포트만 다르게)
```

## 상품 데이터 (mock)

필드: `id, name, category, price, emoji, description, stock`

- 8~16개 정도
- 카테고리 3~4개 (예: 의류, 전자기기, 식품, 뷰티)

## 화면/기능

### 1. 상품 목록
- 카드 그리드 (이미지/이모지, 이름, 가격, "담기" 버튼)
- 상단 검색창 (이름 기준 실시간 필터링)
- 카테고리 탭/버튼 (전체 포함)
- 정렬 드롭다운 (가격 낮은순/높은순)

### 2. 상품 상세
- 카드 클릭 시 모달 또는 별도 섹션으로 확대
- 설명, 재고 상태, 수량 선택 후 담기

### 3. 장바구니
- 우측 슬라이드 패널 또는 별도 화면
- 항목별: 이름, 단가, 수량 +/- 버튼, 삭제 버튼, 소계
- 하단: 총 수량, 총 금액
- `localStorage`에 저장해서 새로고침해도 유지

### 4. 체크아웃 (목업)
- 이름/주소/연락처 입력 폼 (검증만, 실제 전송 없음)
- "주문하기" 클릭 → 주문번호 랜덤 생성 + 완료 메시지 표시 → 장바구니 비우기

### 5. 공통
- 헤더에 장바구니 아이콘 + 담긴 수량 뱃지
- 상단 고정 네비게이션
- 한글 UI 텍스트

## 아직 미구현
프로젝트 폴더는 비어 있으며 위 계획대로 파일을 아직 생성하지 않은 상태.
